3.8 Enlaces

Lo que caracteriza a la web como medio de comunicación es la navegación por medio de hipervínculos (enlaces).
Esta conexión entre archivos permite al usuario elegir el sentido de la navegación.


En HTML5, los enlaces se marcan con la etiqueta <a> </a> y el atributo principal es href="" , donde se escribe la
ubicación del archivo de destino, el cual puede estar en la misma carpeta que el archivo que lo está llamando, en otra carpeta, o en un sitio web.

Dentro de las etiquetas <a href=""></a> se puede colocar cualquier elemento html, el cual funcionará como botón (generalmente se suele colocar un texto o una imagen).
Si es un texto, el navegador por defecto lo muestra en color azul y subrayado.
Si es una imagen, algunos navegadores le suelen agregar un borde azul.

Tipos de enlaces

Existen diferentes tipos de enlaces que detallamos a continuación:

Enlaces externos

Son los enlaces entre diferentes sitios web. Si queremos vincular un sitio web con otro, tenemos que indicar la ruta absoluta del sitio escribiendo la url completa.
La ruta es el valor que se escribe dentro del atributo href="", y se puede clasificar en rutas absolutas o relativas.

Las rutas absolutas, como ya hemos visto, son las que definen la ubicación completa de un archivo en la web, y empiezan desde http://nombredelsitio.com/.../archivo.extensión
Las rutas relativas, en cambio, definen la ubicación del archivo de destino en relación a la ubicación del archivo que lo está llamando.

Pongamos unos ejemplos:

Ruta absoluta:
<a href="https://www.https://www.iesvirgendelcarmen.com/moodle/course/index.php?categoryid=23">Moodle Virgen del Carmen</a>

La cual genera este enlace: Moodle Virgen del Carmen

En algunos casos, se suele usar el atributo target="" , que nos permite abrir el enlace en una pestaña nueva. Por ejemplo,

<a href="http://google.com" target="_blank>Google en una nueva pestaña</a>

Genera este enlace: Google en una nueva pestaña


Enlaces internos entre archivos .html

Son los enlaces principales de un sitio. Si en el archivo index.html, queremos agregar un vínculo a Página de prueba.html, el código sería así:

Ruta relativa:
<a href="../extra/Página de prueba.html">Página de prueba</a>

La cual genera este enlace: Página de prueba


Enlaces internos: anclas

Las anclas son enlaces hacia un punto determinado dentro de un html. Se usa en varios casos, por ejemplo en una
página de preguntas frecuentes, donde las respuestas están desarrolladas debajo de las preguntas (foros con posts de respuestas).
También en los textos largos, al finalizar muchas veces se coloca un botón para subir, lo cual es un ancla.

El punto de destino tiene que estar marcado con el atributo id="algo" , y en el enlace se coloca un numeral (#) seguido del nombre. Por ejemplo:

Inicio de la página
con código:     <a href="#Inicio">Inicio de la página</a><br>

"salta" a este párrafo colocado en el inicio de esta página:

<p id="Inicio">Inicio de la página</p></p>


Enlaces a una dirección de correo

Se puede vincular una dirección de correo para que abra el programa de correo predeterminado. El código sería:

Consúltenos a <a href="mailto:info@iesvirgendelcarmen.com">info@iesvirgendelcarmen.com</a>

Que produce como resultado:   Consúltenos a info@iesvirgendelcarmen.com


Se le puede añadir más contenido al mensaje, añadiendo '?' al final del correo, y separando los siguientes valores con '&'.
Algunos valores que podemos añadir son: 'subject', 'body', 'cc' para copia, 'bcc' para copia oculta...
Veámoslo con el mismo ejemplo de antes:

Consúltenos a <a href="mailto:info@iesvirgendelcarmen.com?subject=Duda&body=Buenos%20Días,%20">info@iesvirgendelcarmen.com</a>

Que produce como resultado:   Consúltenos a info@iesvirgendelcarmen.com

En nuestro ordenador, se nos abriría la ventana del correo electrónico tal que así:

Este recurso está bastante cuestionado, debido a dos razones:
Primero porque muchos usuarios utilizan correo webmail (Gmail, Yahoo, Hotmail) por lo que este enlace no les sirve;
Y segundo porque estamos facilitando a robots que encuentren nuestro correo y lo incluyan en una base de datos para spam.


Enlaces para ampliar una imagen

Es muy común tener en nuestra página web una galería con miniaturas de fotos que vinculan a imágenes ampliadas.
Si combinamos este recurso con código javascript, logramos galerías de fotos muy atractivas. El código HTML sería así:

<a href="img/IES.jpg"><img src="img/IES-miniatura.jpg" alt="Entrada del IES Virgen del Carmen"></a>


Que produciría como salida la imagen en miniatura:    

Donde podemos hacer click y se nos mostrará la imagen ampliada.


Enlaces para descargar archivos

Esto es mas de lo mismo ya visto en los anteriores apartados, puesto que dentro del atributo href="" procedemos a colocar la ruta del archivo.
Puesto que el navegador no puede abrir en su interior archivos tales como .zip o .rar, éste los descarga en el ordenador del usuario.
Veámoslo con un ejemplo:

<a href="img/img.zip">Descarga todas las fotos vistas en esta sección del manual</a>


Generará el enlace siguiente:    Descarga todas las fotos vistas en esta sección del manual
